# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Animes

- `GET /api/animes`
- `POST /api/animes`
- `GET /api/animes/:id`


### UserAnimes

- `GET /api/useranimes`
- `POST /api/useranimes`
- `GET /api/useranimes/:id`
- `PATCH /api/useranimes/:id`
- `DELETE /api/useranimes/:id`



### AnimeLibraries

- `GET /api/animelibraries`
- `POST /api/animelibraries`
- `GET /api/animelibraries/:id`
- `PATCH /api/animelibraries/:id`
- `DELETE /api/animelibraries/:id`

### Reviews
- `GET /api/reviews`
- `POST /api/reviews`
- `GET /api/reviews/:id`
- `PATCH /api/reviews/:id`
- `DELETE /api/reviews/:id`
